# Random Choice Picker

A simple web application that allows users to input tags separated by commas and then randomly selects and highlights one of the tags. Useful for random selection or decision-making.

## Features

- Input tags in the text area, separated by commas.
- Press Enter to clear the input and randomly select/highlight a tag.
- Tags are displayed as stylish elements in the UI.
- Random selection is animated for a visually appealing effect.
